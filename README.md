#  Super Store Sales Analysis Dashboard (Power BI)

##  Project Overview
This project presents an interactive sales analysis dashboard built using Microsoft Power BI based on the Super Store dataset. The dashboard provides insights into sales performance, profit trends, regional performance, and category-wise contribution to support data-driven business decisions.

##  Business Objectives
- Analyze overall sales and profit performance
- Identify top-performing regions, categories, and segments
- Monitor monthly sales trends
- Evaluate product category contribution to revenue
- Detect loss-making areas for improvement

##  Tools & Technologies
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Modeling
- Super Store Dataset

##  Key Insights
- West region generated the highest sales revenue
- Technology category contributed the most profit
- Some sub-categories showed high sales but low profit
- Seasonal sales growth observed in specific months
- Consumer segment contributed the largest share of sales

##  Dashboard Features
- Interactive filters for Region, Category, and Segment
- KPI cards for Sales, Profit, and Quantity
- Monthly sales trend visualization
- Category and sub-category performance comparison
- Regional performance analysis
